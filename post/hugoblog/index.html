[中文] 使用 Hugo 在 github.io 建立一个 Blog

2019-11-19
中文

Contents

    • 安装 Hugo
    • hugo 初始化一个文件夹作为根目录
    • 选一个 hugo 的 theme
    • 本地启动 hugo 查看效果
    • 写一篇文章
    • 生成 public 文件， 用git 推送到 github

安装 Hugo

1
2

brew install hugo
hugo version

hugo 初始化一个文件夹作为根目录

1
2

hugo new site myblog
cd myblog

选一个 hugo 的 theme

在 https://themes.gohugo.io/ 上选一个 去 repo

1

git clone <repo_url>  themes/<theme_name>

我这边用的 theme 是 even 需要多做一步

1
2
3

cd myblog
rm -rf config.toml
cp themes/even/config.toml ./

可以自己修改 config.toml 文件里面的配置

本地启动 hugo 查看效果

1

hugo server -t even --buildDrafts

写一篇文章

1

hugo new post/new_md.md

执行后显示

1

myblog/content/post/home.md created

此时编辑 home.md 文件即可在本地预览到

生成 public 文件， 用git 推送到 github

在 github 上新建 repository 名字和用户名一致后面是 github.io

eg: <user_name>.github.io

1
2
3
4
5
6
7

hugo --theme=even --baseUrl="https://<github_url>.github.io/" --buildDrafts
cd myblog/public
git init
git add .
git commit -m '.'
git remote add origin https://<github_url>.github.io
git push --set-upstream master

此时访问 .github.io 就可以看到 blog 了，我这边的网址是 https://stupid-pao.github.io/
